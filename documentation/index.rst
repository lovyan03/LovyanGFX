#########
LovyanGFX
#########

LovyanGFX is a graphics library for ESP32 (SPI or 8bit Parallel) and ATSAMD51 (SPI). Supported devices include M5Stack, M5Core2, M5StickC, TTGO T-Watchm ODROID-GO, ESP-WROVER-KIT, WioTerminal and more...
